About the template

Superial is a responsive and multipurpose admin theme powered with Twitter Bootstrap v3.3.5 It can be used for for any type of web applications: custom admin panels, admin dashboards, CMS etc.. Superial has a clean and intuitive design which helps you to create an awesome and powerful project. Superial has a huge collection of plugins and UI components and works great on all major web browsers, tablets and phones.
Please note that this documentation is dedicated to the main element of the template. With each version, we will try to develop it. But if you have any questions going beyond what is described here don't hesitate to write to us.

Enjoy using Superial and stay tuned to endless amazing updates!

Main HTML elements

Theme files have almost unchanging structure consisting of:

  1. #sideBarWrapper left navigation menu.
  2. .nawbarMain top navigation.
  3. #mainWrapper main container for html elements.
  4. .fotterWrapper main container for footer.

Working with Gulp

First, you must download and install node.js (which includes npm). npm stands for node packaged modules and is a way to manage development dependencies through Node.js. You can check it in your terminal window using these commands node --version and npm --version.

Installing Gulp

Gulp is a JavaScript task runner (Automation), save yourself from repetitive tasks. It's how we compile our code, run tests, and more.

  • Install gulp globally with npm install --global gulp.
  • Then run npm install. npm will look at the package.json file and automatically install the necessary local dependencies listed there.
  • When completed, you'll be able to run the various Gulp commands provided from the command line.

Gulp Commands

To run gulp use gulp watch command in your command line. You can modify or add your task in gulpfile.js.

Other commands

  • gulp fileinclude will compile all html files

Customize Styles

Using CSS

All style changes, fixes, and improvements made for Superial will typically be added to css folder.

To prevent losing your changes, you should add all of your customizations to a new stylesheet and include it separately in your pages.

Using SASS

SASS is powerful CSS pre-processor system. It will give you more accesibility to customize your styles.

All custom scss styles you can find in build/css/scss folder.

If you want to find some more information about SASS click here.
